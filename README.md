# SHL Assessment Agent

An AI-powered FastAPI application that recommends relevant SHL assessments based on user queries using semantic search and vector similarity.

## Features
- FastAPI REST API
- Semantic search using FAISS
- SHL assessment recommendation system
- Swagger API documentation
- Public cloud deployment using Render

## Tech Stack
- Python
- FastAPI
- FAISS
- Uvicorn
- Render
- GitHub
